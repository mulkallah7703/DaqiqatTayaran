# بنية المشروع - منصة دقيقة طيران (Daqiqat Tayaran)

## Aviation AI Platform - Project Structure & Technology Stack

---

## 1. نظرة عامة على المشروع | Project Overview

**اسم المشروع:** Aviation AI Platform (منصة الطيران الذكية)  
**الوصف:** منصة متكاملة تعمل بالذكاء الاصطناعي تشمل نظام إدارة التعلم (LMS) ولوحة تحكم إدارية  
**الرؤية:** منصة سعودية رائدة في قطاع الطيران تتماشى مع رؤية السعودية 2030

---

## 2. البنية المعمارية | Architecture

```
Dagigattayaran/
├── client/                 # تطبيق الواجهة الأمامية (React + Vite)
├── routes/                 # مسارات API الخلفية
├── models/                 # نماذج قاعدة البيانات (MongoDB)
├── middleware/             # وسائط المصادقة والأمان
├── uploads/                # ملفات SCORM والوسائط
├── server.js               # خادم Express الرئيسي
├── seed.js                 # تهيئة البيانات الأولية
├── Dockerfile              # تكوين النشر
├── docker-compose.yml      # تكوين الحاويات
└── package.json            # تبعيات المشروع الرئيسية
```

## 8. واجهات API | API Endpoints

- **Auth:** `/api/auth/*` - تسجيل الدخول، التسجيل، JWT
- **Users:** `/api/users/*` - إدارة المستخدمين
- **Admin:** `/api/admin/*` - لوحة التحكم
- **Courses:** `/api/courses/*` - إدارة الدورات
- **Content:** `/api/content/*` - إدارة المحتوى
- **AI:** `/api/ai/*` - مساعد الذكاء الاصطناعي

---

## 9. متغيرات البيئة | Environment Variables

### Client (.env)
- `VITE_EMAILJS_SERVICE_ID` - خدمة EmailJS
- `VITE_EMAILJS_TEMPLATE_ID` - قالب البريد
- `VITE_EMAILJS_PUBLIC_KEY` - المفتاح العام

### Server
- `MONGODB_URI` - رابط قاعدة البيانات
- `JWT_SECRET` - مفتاح JWT
- `OPENAI_API_KEY` - مفتاح OpenAI
- `ADMIN_EMAIL` / `ADMIN_PASSWORD` - بيانات الإدارة
- `CLIENT_URL` - عنوان الواجهة الأمامية

## 10. النشر والتشغيل | Deployment

- **Docker:** Dockerfile + docker-compose.yml
- **Build:** `npm run build` (Vite)
- **Start:** `npm start` (Node + Express)
- **Dev:** `npm run dev` (Concurrently: server + client)
